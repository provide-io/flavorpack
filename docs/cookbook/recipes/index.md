## Using These Recipes

Each recipe provides:

1. **Problem statement** - What scenario this solves
2. **Complete solution** - Working code and configuration
3. **Variations** - Different approaches for different needs
4. **Best practices** - Tips from real-world usage

## Recipe vs Example?

**Recipes** are short, focused how-tos for specific integrations.
**Examples** are complete applications showing end-to-end packaging.

Choose recipes when you want to:
- Integrate with a specific tool (Docker, CI/CD)
- Learn a specific technique
- Find a quick solution

Choose examples when you want to:
- See a complete application
- Learn packaging from scratch
- Follow a tutorial
